Info

Quixfer is an hybrid app that allows local money transaction management (offline) and then supports synchronization with Buxfer platform (www.buxfer.com). Quixfer is NOT an official buxfer application, and does not have any relation with buxfer.com owners. Quixfer uses buxfer public API.

Help:

- Use ADD view to add expense (-) and income (+) transactions.

- Use SYNCH view to synchronize added transactions with buxfer platform.

- Use SET view to configure a buxfer user and preferences as transaction description default.

Credits:

Quixfer is built upon AngularJS, Bootstrap and Apache Cordova. Moreover, Quixfer depends on the following components: UI Bootstrap, ngTagsInput, Font-Awesome Icon and Crypto-js.
